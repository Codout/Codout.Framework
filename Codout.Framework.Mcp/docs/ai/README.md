# Codout AI Docs Pack

Este pacote materializa o padrão UI/CRUD do ecossistema Codout em arquivos versionáveis.

## Onde colocar
Copie a pasta `docs/ai` para a raiz do repositório consumidor, por exemplo `Codout.Club/docs/ai`.

## Ordem de uso
1. `constitution/ui-crud.md`
2. `catalog/*`
3. `references/gold/*`
4. `recipes/*`
5. `CLAUDE.md`
6. `mcp/*`

## Objetivo
Servir como base curada para:
- Claude no VSCode
- servidor MCP do ecossistema Codout
- onboarding técnico
- governança de UI
